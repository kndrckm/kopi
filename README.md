# kopiAPK - Monicoffee Android App

An Android application built with Capacitor for tracking coffee consumption, featuring ML Kit-powered background removal for creating custom coffee stickers.

## Features

- **Coffee Tracking**: Log your daily coffee consumption
- **ML Kit Background Removal**: Remove backgrounds from images using Google's ML Kit Subject Segmentation
- **Custom Stickers**: Create personalized coffee stickers with automatic outline and shadow effects
- **Statistics**: View weekly, monthly, and yearly coffee consumption statistics
- **PWA Support**: Progressive Web App capabilities

## Tech Stack

- **Capacitor**: Native Android app wrapper
- **ML Kit Subject Segmentation**: Native background removal via `@capacitor-mlkit/subject-segmentation`
- **Vanilla JavaScript**: No framework dependencies

## Prerequisites

- Node.js (v16 or higher)
- Android Studio (for building and running the Android app)
- Android SDK

## Installation

1. Install dependencies:
```bash
npm install
```

2. Sync web assets with Capacitor:
```bash
npm run sync
```

## Development

### Build the app
```bash
npm run build
```

### Open in Android Studio
```bash
npm run android:open
```

### Run on connected device/emulator
```bash
npm run android:run
```

## Background Removal

The app uses Google's ML Kit Subject Segmentation plugin for native background removal:

1. Images are processed using the `@capacitor-mlkit/subject-segmentation` plugin
2. The subject is segmented from the background
3. A white outline and drop shadow are applied automatically
4. The final sticker is trimmed and optimized

If ML Kit fails, a fallback mechanism is in place.
